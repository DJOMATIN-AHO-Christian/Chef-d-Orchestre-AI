# Chef d'Orchestre AI 

Dirigez un orchestre virtuel avec vos gestes grâce à l'IA et Tone.js.

## Fonctionnalités
- **Mode IA (Caméra)** : Contrôlez la musique par les mouvements de vos mains.
- **Mode Démo (Souris/Tactile)** : Fallback performant si la caméra est indisponible.
- **Moteur Musical Intégral** : Gammes pentatoniques, majeures, mineures, etc.
- **Visualisation Temps Réel** : Analyseur FFT et système de particules.

## Technologies
- **TensorFlow.js / PoseNet** (IA)
- **Tone.js** (Audio)
- **Vanilla JS / CSS** (Interface)

Développé pour une expérience musicale immersive et performante.
